# X-uitter Front

Simple Vue.js proof-of-concept frontend created to consume and validate the X-uitter API MVP.

## Tech Stack

- Vue 3
- TypeScript
- Vite
- Native Fetch API

## Requirements

- Node.js
- npm
- X-uitter API running locally
